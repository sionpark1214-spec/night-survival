# 🌙 밤의 생존 (Night Survival)

모바일 좀비 생존 게임. 낮에 자원과 돈을 모으고, 밤이 되면 사방에서 몰려오는 좀비를 막아라.

## 플레이
- **▶ 바로 플레이**: https://sionpark1214-spec.github.io/night-survival/
- 단일 `index.html` — 받아서 브라우저로 열어도 됨

## 조작 (모바일)
- 왼쪽 스틱 이동 / 오른쪽 스틱 조준+자동발사
- 🔪/🔫 무기 전환, 🎒 인벤토리
- 건물(총 가게·고용 가게·분쇄작업대) 앞에 가면 하단 메뉴

## 핵심 시스템
- **온도 & 체력**(둘 다 0이면 사망), 캠프파이어 곁에서 회복
- 낮(20초~, 하루마다 +30초) → 밤(60초, 좀비) → 아침 반복, 무한 생존
- 총 9종(권총~미니건), 좀비 8종(일반~보스)
- 좀비 처치 5%로 철·돌·나무 드롭 → **분쇄작업대**에서 지뢰·터렛·용암함정·벽·횃불 제작
- **고용**: 칼든사람·경찰·저격수·의무병·광부 (최대 30, 따라다니며 자동 전투)

PC에서는 WASD + 마우스로도 플레이 가능.
